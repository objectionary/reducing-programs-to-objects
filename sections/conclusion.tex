\section{Conclusion}

We demonstrated how some language features often present in high-level
  object-oriented languages, such as Java or C++, may be expressed
  using objects.
We used \eolang{} as a destination programming language because of its
  minimalistic semantics: it has no language features aside from objects
  and their composition and decoration mechanisms.
