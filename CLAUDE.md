# Reducing Programs to Objects

## Bibliography

The `bibliography/` directory is a Git submodule pointing at
  [yegor256/bibliography](https://github.com/yegor256/bibliography).
Never edit `bibliography/main.bib` directly in this repository.
To add or change a reference, submit a pull request to the
  `yegor256/bibliography` repository, wait for it to be merged, and then
  pull the updated submodule here.
